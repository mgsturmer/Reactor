Privacy Policy

This app, Reactor, does not collect, store, or share any personal data from users.

The only data used is the device’s approximate location, solely to determine the app's interface language (Portuguese or English). No data is sent to external servers.

All gameplay data (such as reaction times and game results) is stored locally on the user’s device. The app does not connect to the internet to transfer, save, or process personal information.

We do not use cookies, trackers, or third-party libraries for data collection. This application is intended solely for entertainment purposes.

If you have any questions, please contact us via email: mgsturmer@gmail.com

Política de Privacidade

Este aplicativo, Reactor, não coleta, armazena ou compartilha dados pessoais dos usuários.

O único dado utilizado é a localização aproximada do dispositivo, com o propósito exclusivo de definir automaticamente o idioma da interface (português ou inglês). Nenhum dado é transmitido a servidores externos.

Todos os dados de jogo (como tempos de reação e resultados de partidas) são armazenados localmente no dispositivo do usuário. O app não se conecta à internet para transferir, salvar ou processar dados pessoais.

Não utilizamos cookies, rastreadores ou bibliotecas de terceiros para coleta de informações. Este aplicativo é voltado exclusivamente para fins de entretenimento.

Em caso de dúvidas, entre em contato pelo e-mail: mgsturmer@gmail.com
